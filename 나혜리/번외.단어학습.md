## 단어 학습

| 용어 | 설명 |
|---|---|
| **린트(Lint) / 린터(Linter)** | 소프트웨어 정적 분석 도구로, 프로그램의 오류·버그·스타일 문제 등을 찾아 표시하는 자동화 도구 |
| **린팅(Linting)** | 린트를 사용해 코드의 품질과 규약을 검증하는 일련의 과정 |
| **에이전틱 AI** | 스스로 문제를 분석하고 도구를 활용하여 자율적으로 작업을 수행하는 AI 시스템. 작업을 제안하는 데서 끝나지 않고 실행까지 완수하는 것이 특징 |
| **MCP (Model Context Protocol)** | AI 모델이 외부 데이터 소스 및 도구와 상호작용할 수 있도록 표준화된 프로토콜 |
| **컨텍스트 윈도우** | 한 번의 추론에서 참조할 수 있는 토큰의 총량 |
| **claude doctor** | 시스템 상태를 진단하고 문제를 해결하기 위해 상태값을 불러오는 명령어 |
| **테스트 스캐폴딩** | 테스트 코드의 초기 구조를 자동 생성하는 기법 |
