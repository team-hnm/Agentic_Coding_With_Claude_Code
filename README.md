## 💡 진행 방식
- **매주 한 챕터**씩 공부한 후 **목요일 23시 59분 59초**까지 각자의 폴더에 내용을 정리한다.
- 읽으며 궁금했던 **질문 1~3개**를 **이슈에 등록**한다.
- **매주 금요일,** 이슈에 등록된 질문들을 중심으로 공통 회고록을 정리한다.

## ⛔️ 스터디 주의 사항
- 책 내용을 그대로 옮겨 적는 것이 아닌 이해한 내용을 위주로 **자신만의 언어로 정리**한다.
- **'모르겠어요'보다는 토론할 수 있는 방식의 질문을 추구합시다!**
- 등록된 이슈에 대해서는 늘 댓글이 열려있으니 자유로운 토론을 추구합시다!

## ✍🏻 스터디 방식
1. 해당 레파지토리를 clone 하세요
2. main 브랜치에서 개인 아이디의 이름을 가진 브랜치를 생성하여 주세요.
3. 개인 브랜치에 본인의 폴더에 정리한 내용을 push 해주세요. ex) ch01 클로드 코드 시작하기.md
4. 개인 브랜치에서 main 브랜치로 PR(Pull Request)를 보내주세요. ex) docs: 000 챕터 00내용 학습 
5. PR은 금요일 오전 중에 merging됩니다.
